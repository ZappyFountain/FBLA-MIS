# FBLA MIS Study Tool - Official-Modeled Question Bank

This package contains a new 1,000-question `questions.js` bank modeled on the uploaded FBLA sample questions and the 2025-2026 MIS event guidelines.

Upload `index.html` and `questions.js` together to the root of your GitHub Pages repository.

Distribution:
- Systems Design and Analysis: 250
- OOP Concepts: 150
- Data and Information Management: 150
- IT Project Management: 150
- IT Infrastructure: 150
- Emerging Business Technologies: 150

The questions are original practice items designed to match the sample style: short scenario stems, business contexts, four answer choices, and concise explanations.

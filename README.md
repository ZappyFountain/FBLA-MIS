# FBLA MIS Study Tool

This GitHub Pages-ready study tool includes 1,000 MIS questions and three local student profiles: **Zain**, **Ethan**, and **Yohann**.

## How progress works
- Select a student from the top-right Student menu.
- Each profile automatically saves its current session, answered questions, confidence/rationale entries, review queue, and lifetime topic statistics.
- Data is stored only in the browser/device using local storage. It does not sync between teammates' computers, browsers, or incognito sessions.

## GitHub Pages
Upload `index.html`, `questions.js`, and this README to the repository root. Then configure **Settings → Pages → Deploy from branch → main → /(root)**.

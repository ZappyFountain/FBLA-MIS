# FBLA MIS Study Tool — 1,000-question bank

## Files
- `index.html`: GitHub Pages-ready static website.
- `questions.js`: 1,000-question bank. Keep this file in the same folder as `index.html`.

## Deployment
Upload both files to the root of the same GitHub repository. In GitHub, enable Pages from `main` / `(root)`.

## Notes
- The rationale-required mode records the student's explanation before showing the answer. It intentionally does not send any data to an AI API.
- Review-queue data is saved only in the current browser through local storage.
- The question bank is balanced to mirror the event blueprint: 250 Systems Design & Analysis questions and 150 each for OOP, Data & Information Management, IT Project Management, IT Infrastructure, and Emerging Technologies.
